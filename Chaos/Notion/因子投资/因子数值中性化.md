### 为什么 Alpha101 的很多因子（尤其是 alpha_001）最后要“- 0.5”？

在 Alpha #001 的公式末尾有这一步：

```plain text
rank(...) - 0.5

```

把排名结果从 [0, 1] 区间转换成 [-0.5, 0.5] 区间。

这个操作不是随便减的，它有明确的量化金融意义，主要目的是实现**市场中性（market-neutral）**，让因子信号更适合构建**多空对冲策略**。下面详细解释“有啥说法”：

### 1. **让因子值围绕 0 对称（零均值）**

- rank() 后得到的值在 0 到 1 之间，平均值是 0.5。
- 减 0.5 后，平均值变成 0（全市场均值 ≈ 0）。
- 好处：
    - 因子本身不带系统性方向偏差。
    - 当你用这个因子排序选股时，正值自然对应“做多”，负值对应“做空”，不需要额外判断方向。

### 2. **方便构建多空等权对冲组合**

Alpha101 系列因子最初就是为**多空对冲基金**（long-short hedge fund）设计的典型信号：

- 因子值 > 0 → 做多这只股票。
- 因子值 < 0 → 做空这只股票。
- 减 0.5 后，因子值天然以 0 为分界线：
    - 最高 0.5（最强多头）
    - 最低 -0.5（最强空头）
- 如果你等权做多前 30% 股票、等权做空后 30% 股票，整个组合的净曝光接近 0（市场中性），只赚取个股相对强弱的超额收益，而不赌市场整体涨跌。

### 3. **与传统动量/反转因子的习惯一致**

- 很多经典因子（如短期反转、动量）在处理后也喜欢做成“零均值”。
- 减 0.5 是最简单的一种中性化方式（比市值中性、行业中性更基础）。
- WorldQuant 的 101 个因子大部分都带这一步（或类似 scale(-1,1) 等操作），统一风格，便于后续多因子合成时直接加权平均。

### 4. **实际效果对比**

假设不减 0.5（保持 0~1）：

- 所有股票因子值都是正的 → 你只能做多，无法自然生成空头信号。
- 组合会有净多头曝光，收益会严重受市场 beta 影响（牛市赚大钱，熊市亏大钱）。

减了 0.5 后：

- 正负明确，多空平衡。
- 策略收益更接近“纯 alpha”（选股能力），而非市场方向。

### 总结一句话说法

**“-0.5”是为了把截面排名转换成一个市场中性的多空信号，让因子值以 0 为中心，正的做多、负的做空，从而天然适合构建多空对冲组合，剥离市场整体涨跌的影响，只捕捉个股相对强弱。**

这也是为什么 Alpha101 系列因子在对冲基金和量化平台上特别流行——它们天生就是“long-short ready”的信号。

如果你把 alpha_001 的值直接拿来排序，前一半（>0）做多、后一半（<0）做空，就是一个最基础的多空策略雏形了。
