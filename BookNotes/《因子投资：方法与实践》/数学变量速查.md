
## 一、收益率相关变量：你的目标函数

| 变量符号           | 专业名称                     | 详细解释（通俗含义）                              |
| :------------- | :----------------------- | :-------------------------------------- |
| **$R_i$**      | **资产收益率**                | 股票 $i$ 在某个时期的总收益。                       |
| **$R_f$**      | **无风险收益率**               | 钱躺在银行或国债里的收益（基准）。                       |
| **$R_M$**      | **市场组合收益率**              | 整个大盘（如沪深 300）的收益。                       |
| **$R_i^e$**    | **超额收益 (Excess Return)** | $R_i - R_f$。量化研究的标配，即你比存银行多赚了多少钱。       |
| **$E[\cdot]$** | **期望符号**                 | 代表“长期平均”或“未来预测”。$E[R_i^e]$ 就是我们预期的超额收益。 |

## 二、风险暴露相关变量：资产的“标签”

这些变量回答了：这只股票到底是什么属性？它对什么敏感？

| 变量符号 | 专业名称 | 详细解释（通俗含义） |
| :--- | :--- | :--- |
| **$\beta_i$ (Beta)** | **因子暴露 / 载荷 (Loading)** | **它是资产的属性。** 衡量资产对某个因子的敏感度。 |
| **$\boldsymbol{\beta}_i$ (加粗)** | **因子暴露向量** | 如果有多个因子，$\boldsymbol{\beta}_i$ 就是一串数字。比如 `[1.2, -0.5, 0.8]`，分别代表它在市场、规模、价值因子上的暴露。 |
| **$\boldsymbol{\beta}$ (矩阵)** | **因子暴露矩阵** | 全市场 $N$ 只股票在 $K$ 个因子上的暴露，是一张巨大的表格。 |

---

## 三、因子收益相关变量：风险的“价格”

这些变量回答了：如果你承担了某种风险，市场给你多少奖金？

| 变量符号 | 专业名称 | 详细解释（通俗含义） |
| :--- | :--- | :--- |
| **$\lambda$ (Lambda)** | **预期因子溢价 (Premium)** | **它是因子的单价。** 代表承担“单位因子暴露”理应获得的预期回报。 |
| **$\hat{\lambda}$** | **因子收益率的估计值** | 通过回归算出来的那个“平均收益”。 |
| **$\lambda_t$** | **因子在 $t$ 时刻的收益** | 因子也是波动的，这代表 $t$ 月该因子的表现（如：这个月小盘股确实跑赢了大盘股）。 |
| **$\boldsymbol{\lambda}$ (向量)** | **因子收益向量** | 所有 $K$ 个因子各自对应的溢价。 |

## 四、误差与异象相关变量：模型的“漏洞”

| 变量符号 | 专业名称 | 详细解释（通俗含义） |
| :--- | :--- | :--- |
| **$\alpha_i$ (Alpha)** | **定价误差 / 异象 (Anomaly)** | 模型解释不了的“多余收益”。如果 $\alpha$ 显著为正，说明你发现了财富密码。 |
| **$\varepsilon_{it}$ (Epsilon)** | **残差 / 随机扰动** | **由于运气或个股新闻**导致的乱动。这部分波动是任何因子都解释不了的（比如公司董事长突然被抓）。 |
| **$u_{it}$** | **特质性收益** | 与 $\varepsilon$ 含义相近，指剔除因子影响后，属于该股票特有的收益部分。 |

---

## 五、风险与协方差相关变量：降维的密码

| 变量符号 | 专业名称 | 详细解释（通俗含义） |
| :--- | :--- | :--- |
| **$\Sigma$ (Sigma)** | **资产协方差矩阵** | 衡量所有股票两两之间如何同步跌涨。这是个 $3000 \times 3000$ 的大怪兽，计算量巨大。 |
| **$\Sigma_\lambda$** | **因子协方差矩阵** | 衡量几个因子之间如何同步跌涨。这只是个 $10 \times 10$ 的小矩阵，计算非常快。 |
| **$\Sigma_\varepsilon$** | **特质风险矩阵** | 衡量个股自己乱动的方差。通常认为股票间互不干扰，所以是个只有对角线有数字的矩阵。 |

## 六、进阶与方法论相关变量：高阶工具

| 变量符号 | 专业名称 | 详细解释（通俗含义） |
| :--- | :--- | :--- |
| **$m_{t+1}$** | **随机折现因子 (SDF)** | GMM 的核心。代表了投资者在不同经济状态下的“痛苦程度”。 |
| **$\gamma$ (Gamma)** | **截面回归的截距项** | 在截面回归中，代表不暴露在任何因子上时的基础收益。 |
| **$\rho_s$ (Rho)** | **Spearman 秩相关系数** | 用于单调性检验，看因子的 10 个组是不是排队上升的。 |
| **$T, N, K$** | **维度指标** | **$T$**: 历史天数；**$N$**: 股票总数；**$K$**: 因子总数。 |

## 💡 核心逻辑串联：

当你看到这个公式： $E[R_i^e] = \alpha_i + \boldsymbol{\beta}_i' \boldsymbol{\lambda}$

1.  **你想赚的钱** ($E[R_i^e]$) 分为两块：
2.  **靠本事/运气多赚的** ($\alpha_i$)。
3.  **靠承担风险换来的** ($\boldsymbol{\beta}_i' \boldsymbol{\lambda}$)。
    *   $\boldsymbol{\beta}_i$ 是你的**持仓风格**（比如你偏爱小盘股）；
    *   $\boldsymbol{\lambda}$ 是这种风格**在市场上的赏金**（小盘股溢价）。

当你看到这个公式： $\boldsymbol{\Sigma} = \boldsymbol{\beta} \boldsymbol{\Sigma}_\lambda \boldsymbol{\beta}' + \boldsymbol{\Sigma}_\varepsilon$

1.  **全市场的总风险** ($\Sigma$) 太难算。
2.  因子模型把它拆成：**因子的波动** ($\boldsymbol{\Sigma}_\lambda$) 传导给股票 + **股票自己的乱动** ($\Sigma_\varepsilon$)。
3.  这就是**风险降维**的本质。
