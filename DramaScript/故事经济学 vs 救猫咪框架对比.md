# 故事经济学 vs 救猫咪框架对比

## 一句话概括

- **《救猫咪》**：编剧的**施工图纸**——告诉你故事的骨架长什么样，每一块骨头放在哪里
- **《故事经济学》**：故事的**底层操作系统**——告诉你故事为什么能打动人，以及如何把故事变成商业武器

## 维度对比

| 维度 | 救猫咪 | 故事经济学 |
|------|--------|-----------|
| **作者背景** | 编剧（Blake Snyder） | 编剧大师 Robert McKee + 营销专家 |
| **核心问题** | "故事怎么写？" | "故事为什么有效？怎么用它赚钱？" |
| **方法论** | 15个节拍表（结构模板） | 8阶段故事设计（原理框架） |
| **关注重点** | 时间线上的**节奏**和**位置** | 价值观的**正负摆动**和**情感动态** |
| **颗粒度** | 精确到"第几页该发生什么" | 聚焦"为什么这个转折能打动人" |
| **适用场景** | 电影剧本、长篇故事 | 电影 + **营销、广告、品牌故事** |
| **对冲突的理解** | 外部事件驱动（反派逼近、失去一切） | 四层冲突模型（物理/社会/个人/内在） |
| **对角色的要求** | "救猫咪"让观众喜欢主角 | "感同身受"比"讨人喜欢"更重要 |
| **终极目标** | 写出结构完整的好故事 | 让故事驱动**行为**（购买、传播、认同） |

## 核心差异详解

### 1. 结构 vs 原理

救猫咪给你一张清单：开场画面→阐明主题→布局铺垫→触发事件→展开讨论→进入第二幕→副线故事→玩闹游戏→中点→反派逼近→失去一切→灵魂黑夜→进入第三幕→结局→终场画面。照着填就行。

故事经济学告诉你：故事的本质是**核心价值观在正负之间的摆动**。每一次摆动产生**转折点**，转折点制造**悖反期望**，悖反期望引发**情感**。它不给你模板，给你的是引擎原理。

### 2. "怎么摆" vs "为什么动"

救猫咪说：中点要放在故事正中间，是"虚假的胜利"或"虚假的失败"。

故事经济学说：转折点之所以有效，是因为它同时产生**理性效应**（观众回溯寻找原因→顿悟）和**情感效应**（价值正负切换→化学物质注入血液）。情感是变化的副产品，变化停止，情感消散。

### 3. 商业延伸

救猫咪止步于"写出好故事"。

故事经济学多走一步：**目的型故事**。它提出了"镜像体验"理论——消费者认同主角→潜意识互换（用自己的欲望替代角色欲望）→重新演绎（购买产品来重温故事体验）。品牌忠诚度就是这么来的。

### 4. 类型系统

救猫咪有10种故事类型（屋里有怪物、金羊毛、傻瓜获胜等），每种类型有固定的3要素公式。

故事经济学没有类型分类，而是用**核心价值观**来区分故事：爱/恨、真实/谎言、生/死……不同的核心价值观决定不同的情感体验。

## 怎么一起用？

最佳实践是**故事经济学打地基，救猫咪搭框架**：

1. 先用故事经济学确定：目标受众是谁？核心价值观是什么？欲望对象是什么？要制造什么情感动态？
2. 再用救猫咪的15节拍表排布具体情节：哪里放触发事件、哪里放灵魂黑夜、哪里放高潮
3. 写的过程中用故事经济学检验：每个转折点是否制造了悖反期望？价值观是否在正负之间持续摆动？情感动态是否递进？

**一个管"为什么有效"，一个管"怎么排列"，互补而非互斥。**
