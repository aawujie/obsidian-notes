
> 基于《借势》金枪大叔营销方法论 + 《张策短视频笔记》内容创作方法论设计

## 一、人设定位

### 核心人设：肌肉 × 抽象 = 反差炸裂

> "不协调才是潮，才是少年叫板中年的资本。" ——《借势》

| 维度   | 设定                                            |
| ---- | --------------------------------------------- |
| 视觉锤  | 一身腱子肉 + 永远穿不搭的衣服（西装配人字拖、紧身背心配花棉裤），发型随机，表情永远认真 |
| 文字钉  | **"练的不是肌肉，练的是脑回路"**                           |
| 人格标签 | 健身博主的身体 + 哲学家的灵魂 + 小学生的逻辑                     |
| 说话风格 | 一本正经地胡说八道，把荒诞的事用极其严肃的语气讲出来                    |

### 为什么这个人设能火？（对照《借势》方法论）

1. **足够叛逆，反对你的人足够多** —— 健身圈觉得他不正经，搞笑圈觉得他太壮，两边都看不惯 = 两边都来看
2. **巨大的缺陷 = 风格 = 疯狂的粉丝 = 消费购买力** —— 他的"抽象"就是缺陷，但正是这个缺陷让人上瘾
3. **知行合一：面孔、语言、行为同时指向一个特质** —— 肌肉是视觉锤，抽象是行为锤，两者绑定不可分割
4. **品位是流量的敌人** —— 不端着，不装逼，90%的用户看得懂、笑得出

### 需要弥补的「六大缺憾」（人物弧光设计）

1. 只会练不会吃（经常吃垃圾食品被粉丝骂）
2. 社交能力为零（跟人说话永远冷场）
3. 穿搭灾难（但他自己觉得很帅）
4. 做饭能把厨房炸了
5. 恋爱经验为零（相亲系列天然内容矿）
6. 文化水平感人（但偶尔蹦出一句深刻的话）

> 这些缺陷不是bug，是**内容矿脉**。每一个缺陷都是一个系列选题。

## 二、内容矩阵

### 四大内容支柱

> "你的产品是否具有社交性？没有社交性就无法产生裂变。" ——《借势》

| 支柱 | 内容形式 | 社交货币 | 更新频率 |
|---|---|---|---|
| **抽象健身** | 用荒诞方式教真实动作（如：用龙虾姿势练背阔肌） | 转发搞笑 + 真能学到东西 | 每周3条 |
| **一本正经生活** | 做饭、穿搭、相亲，全程严肃脸做抽象事 | "这人有病吧哈哈哈" | 每周2条 |
| **肌肉哲学** | 健身间隙突然输出一段人生感悟，似是而非但细想有道理 | 截图传播，当签名用 | 每周1条 |
| **挑战/联动** | 跟其他博主互动，用肌肉解决一切问题 | 粉丝互导 | 每月2-3条 |

### 内容创作公式

> "成为传播大师，只需要借助四点：痛点、笑点、泪点、尿点" ——《借势》

**每条视频 = 1个笑点 + 1个知识点 + 1句金句**

- **笑点**：抽象行为（用蛋白粉泡茶、穿西装做深蹲）
- **知识点**：真实有效的健身/生活技巧（藏在抽象里）
- **金句**：结尾一句看似胡说实则有理的话（传播钩子）

### 标题/封面策略

> "粉丝少，更要花大量时间去起标题" ——《张策短视频笔记》

- 封面：永远是陈泽峰**认真脸 + 荒诞场景**的对比
- 标题公式：**悬念 + 反差**（"练了三年胸肌，只为了夹住这根筷子"）

## 三、平台策略

> "短视频作为流量平台的四大卖点：创作平台、消费者调研平台、打造IP、试错平台" ——《借势》

| 平台 | 定位 | 内容侧重 |
|---|---|---|
| **抖音** | 主战场，冲流量 | 15-60s 抽象健身短视频，强节奏、强反转 |
| **B站** | 深度内容，养铁粉 | 3-8分钟 vlog，展示完整的抽象日常 |
| **小红书** | 种草变现 | 穿搭翻车合集、健身餐（黑暗料理）图文 |
| **视频号** | 下沉市场 | 搬运抖音爆款，触达中年用户（他们最爱转发） |

### 起号节奏

1. **0-1万粉**：每天发，测试哪类抽象内容数据最好（试错平台）
2. **1-10万粉**：固定栏目化，形成"每周三抽象健身日"等用户期待
3. **10-50万粉**：开始联动，借势大博主（贴、靠、要）
4. **50万+**：开始商业化，但**内容质量不能降**

## 四、品牌故事

> "写出一个品牌故事只需要三步：卖惨、绝处逢生、大爱洒向人间" ——《借势》

### 陈泽峰的品牌故事

**卖惨**：陈泽峰，一个从小被嘲笑"竹竿"的瘦弱男生。高中被霸凌，大学社恐到不敢去食堂。他决定健身改变自己，但因为太穷买不起蛋白粉，只能每天吃六个鸡蛋白。

**绝处逢生**：三年后，他练出了一身腱子肉。但他发现，自己变壮了，性格却没变——还是那个说话冷场、穿搭灾难、做饭能炸厨房的社恐男。直到有一天，他把自己"一本正经做抽象事"的日常发到网上，评论区炸了。

**大爱洒向人间**：他发现，原来这个世界上有那么多和他一样"不合群"的人。他决定用自己的方式告诉所有人：**你不需要变成别人眼中的正常人，你只需要把自己的"不正常"做到极致。**

> "每个品牌都应该找到自己的至暗时刻，如果实在没有，可以自己创造一个。" ——《借势》

## 五、商业变现闭环

> "在一个充满竞争的市场，想要翻盘的三个方法：产品力是里子，品牌力是面子，影响力才是票子。" ——《借势》

### 变现五层飞轮

```
内容获客 → 人设信任 → 产品种草 → 私域转化 → 复购裂变
   ↑                                              |
   └──────────── 用户反馈驱动新内容 ←──────────────┘
```

### 具体变现路径

| 层级 | 变现方式 | 预计时间 | 月收入预估 |
|---|---|---|---|
| **第一层：广告** | 品牌植入（运动品牌、食品、日用品） | 5万粉起 | 2-5万 |
| **第二层：自有产品** | "泽峰严选"蛋白粉/健身零食（抽象包装） | 20万粉起 | 5-15万 |
| **第三层：知识付费** | "抽象健身课"——用最搞笑的方式教最科学的训练 | 30万粉起 | 3-8万 |
| **第四层：线下** | "泽峰抽象健身房"快闪店/联名活动 | 50万粉起 | 单场5-20万 |
| **第五层：IP授权** | 表情包、联名款、动画形象 | 100万粉起 | 长尾收入 |

### 产品设计原则

> "你的产品必须有说头、有社交性、有娱乐性" ——《借势》

**自有产品必须满足三个条件：**

1. **有说头**（知识性）：蛋白粉包装上印"泽峰冷知识"，每袋一个健身小贴士
2. **有拍头**（社交性）：包装设计抽象到用户忍不住拍照发朋友圈（丑到极致就是美）
3. **有笑头**（娱乐性）：产品名字本身就是梗（如"脑回路增肌粉""社恐专用摇摇杯"）

### 定价策略

> "品位是流量的敌人，大众品牌要做的第一件事就是先活着。" ——《借势》

- **不走高端**，走"中等收入社交货币"路线
- 核心价格带：**39-129元**（让粉丝买得起、愿意晒）
- 偶尔出限量款（饥饿营销），价格翻倍，制造稀缺感

## 六、粉丝运营

> "当你有了巨大的缺陷，有很多人讨厌你，你就有了风格；有了强烈的风格，就有了疯狂的粉丝。" ——《借势》

### 粉丝分层运营

| 层级 | 人群 | 运营方式 |
|---|---|---|
| **路人粉** | 刷到觉得好笑，随手关注 | 持续产出爆款内容，保持曝光 |
| **活跃粉** | 每条必看，评论区常客 | 翻牌回复，制造"泽峰语录"让他们传播 |
| **铁粉** | 买周边、进社群、线下见面 | 私域社群（微信群），专属内容，新品试用官 |
| **黑粉** | 骂他抽象、骂他不正经 | **不删不怼，让黑粉帮你免费传播** |

> "挑刺的黑粉，会不遗余力地寻找你的漏洞，他比最优秀的市场调研部门还厉害" ——《借势》

### 社群设计

- 群名：**"泽峰抽象研究所"**
- 入群门槛：发一张自己最抽象的健身照（制造门槛 = 制造流行）
- 群内活动：每周一次"抽象健身挑战"，最抽象的送签名周边

## 七、借势策略

> "心态要平，脸皮要厚。贴上去、靠上去硬要，他们不会跟你计较。" ——《借势》

### 短期借势（蹭热点）

- 每个健身圈热点事件，陈泽峰都用抽象方式复刻一遍
- 每个社会热梗，都用"肌肉男一本正经演绎"的方式二创
- **热点 + 反差 = 病毒传播**

### 中期借势（联动）

- 找**反差最大**的博主联动：美妆博主、知识区博主、做饭博主
- 核心逻辑：**陈泽峰出现在任何不该出现的场景里，本身就是内容**

### 长期借势（绑定品类）

- 目标：让"抽象健身"这个品类 = 陈泽峰
- 方法：持续输出，占领用户心智，让后来者都被叫"模仿泽峰的"

## 八、风险控制

> "看透了周期，你就把握了命运。" ——《借势》

| 风险 | 应对 |
|---|---|
| 审美疲劳 | 每季度迭代一个新栏目，保持新鲜感（转折，转折，再转折） |
| 人设翻车 | 人设本身就是"不完美"，翻车空间极小；真出事就正面道歉，符合人设 |
| 抽象过度 | 每条视频必须有一个**真实的知识点**，不能纯搞笑，否则没有复看价值 |
| 变现伤粉 | 广告植入必须"抽象化"，让广告本身成为内容的一部分，粉丝看广告也在笑 |
| 平台政策 | 多平台分发，不把鸡蛋放一个篮子里 |

## 十、一句话总结

> "共识是偏见的终局，一旦事物到了终局，生命就结束了。要不停地在共识中寻找偏见，偏见才是生命力。" ——《借势》

所有人都觉得健身博主应该阳光、自律、正能量。

**陈泽峰偏不。**

他用一身肌肉证明自律，用一脑子抽象证明自由。

**这就是偏见，这就是生命力。**
