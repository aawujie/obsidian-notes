# 抽象向量空间

来源：3Blue1Brown《线性代数的本质》第 15 集（UP 主汉语配音版），Whisper 转录稿整理

---

## 一、向量是什么？

第一期的问题：向量是什么？二维向量可以是平面内的箭头、一组坐标（实数对）、或两者都是更深层次东西的体现。

- **坐标观点**：清晰明了，高维向量像可操作的具体概念；但高维空间只是模糊的几何概念
- **实践者观点**：所处理的空间独立于坐标存在，坐标描述有些随意（依赖所选基向量）。行列式、特征向量等似乎不受坐标系影响——它们暗含于空间中的性质

若向量根本上并不是一组数字，其本质更具空间性。那么数学家所说的「空间」是什么意思？

---

## 二、函数作为向量

函数既不是箭头也不是一组数字，但同样具有向量特性。

**函数加法**：(f + g)(x) = f(x) + g(x)，与向量对应坐标相加类似，只是有无穷多个「坐标」

**函数数乘**：输出值与数相乘，与向量对应坐标数乘类似

因为对向量所能进行的操作不过是相加和数乘，所以以箭头为背景发展的线性代数概念和手段，应能原封不动地应用于函数。

---

## 三、函数的线性变换：求导

函数的线性变换：接收一个函数，输出另一个函数。**求导**就是一个例子（有时称为算子而非变换，意思相同）。

**线性**的严格定义（可加性 + 成比例）：
- **可加性**：L(v + w) = L(v) + L(w)
- **成比例**：L(cv) = c·L(v)

即线性变换保持向量加法和数乘运算。前几期讨论的「网格线保持平行且等距分布」只是这两条性质在二维空间特殊情况下的体现。

**重要推论**：线性变换可通过它对基向量的作用完全描述，这使得矩阵向量乘法成为可能。这一点对函数同样正确。

**求导具有可加性和成比例性**：
- (f + g)' = f' + g'
- (cf)' = c·f'

## 四、用矩阵描述求导

限制在**多项式空间**（如 x² + 3x + 5，4x⁷ - 5x² 等）。每个多项式有限项，但空间包含任意高次多项式。

**赋予坐标**：取 x 的不同次幂作为基函数——b₀(x)=1, b₁(x)=x, b₂(x)=x², b₃(x)=x³, … 基函数的作用与 î、ĵ、k̂ 在箭头世界中类似。因多项式次数可任意高，基函数集无穷大，即多项式作为向量时有**无穷多个坐标**。

**例**：x² + 3x + 5 的坐标 = [5, 3, 1, 0, 0, …]；4x⁷ - 5x² 的坐标 = [0, 0, -5, 0, 0, 0, 0, 4, 0, …]

**求导矩阵**：无穷阶矩阵，绝大部分为 0，次对角线上按序排列正整数（1, 2, 3, 4, …）。矩阵向量乘法的作用：结果的常数项 = 1×原式 x 的系数；x 的系数 = 2×原式 x² 的系数；x² 的系数 = 3×原式 x³ 的系数……与求导规则一致。

求导满足线性性质使得这一过程成为可能。**矩阵向量乘法和求导看似毫不相干，但它们其实是一家人**。点击、特征向量等在函数世界中都有直接类比（内积、特征函数等）。

## 五、向量空间的公理化

数学中有很多类似向量的事物。只要处理的对象具有合理的数乘和相加概念——不管是箭头、一组数、函数的集合，还是你定义的其他东西——线性代数中关于向量、线性变换等的概念都应适用于它。

数学家希望定义和发现不只对一种情况成立，而是对**所有类似向量的事物**都有普适性。这些事物构成的集合被称为**向量空间**。

**公理**：建立向量加法和数乘必须遵守的规则。现代线性代数理论中，向量空间需满足**八条公理**，以保证加法和数乘确实如我们所愿。公理并非自然法则，而是**媒介**——连接发现结论的数学家与想把这些结论应用于新向量空间的人。

若有人定义了奇怪的向量空间（如所有派生物的集合），在应用线性代数结论之前，需验证其加法和数乘是否满足公理。数学家只需根据公理证明结论，只要定义满足公理，结论就能应用——即便数学家从未想过那种情况。

---

## 六、抽象表述的原因

结论往往**抽象表述**——仅根据公理表述，而不集中于某种特定向量（箭头、函数等）。

这就是为什么教科书用**可加性和成比例**定义线性变换，而不是用「网格保持平行且等距分布」——后者更直观，但前者具有一般性，适用于所有满足公理的向量空间。

---

## 七、向量是什么？数学家的回答

**形式并不重要**。箭头、一组数、函数、派生物等都无所谓，可以是任何东西，只要加法和数乘遵守公理即可。

就像问「三究竟是什么」：在具体情境中它代表三个东西的集合，但在数学里它被看作所有「三个东西的集合」的抽象概念。向量也是如此——有多种体现，数学家把它抽象成**向量空间**这一无形概念。

**学习建议**：从具体形象（如二维箭头）出发理解更好；但学到更多后，会知道这些概念有更广泛的应用，这就是教科书倾向于抽象表述的根本原因。
